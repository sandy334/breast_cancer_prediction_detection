Breast Cancer Detection - Model Overview

About the Dataset

This model is trained on the Breast Cancer Wisconsin (Diagnostic) Dataset, containing 569 samples with 30 numerical features from cell nuclei in digitized images. It predicts whether a tumor is malignant or benign.

Feature Information

Key Descriptions:

Radius: Distance from center to edge.
Texture: Pixel intensity variation.
Perimeter: Outline length.
Area: Size of the tumor.
Smoothness: Edge regularity.
Compactness: Shape tightness.
Concavity: Indentations in contour.
Concave Points: Number of concave parts.
Symmetry: Shape balance.
Fractal Dimension: Border complexity.
  • Radius (mean, se, worst)
  • Texture (mean, se, worst)
  • Perimeter, Area, Smoothness
  • Compactness, Concavity
  • Concave Points, Symmetry
  • Fractal Dimension

Model Prediction

The model uses a combination of the 30 features. After training, it learns patterns that help distinguish between benign (0) and malignant (1) tumors. New input data is then used for prediction using this learned knowledge.

Feature Importance

Most Influential Features:

  • Worst Radius & Worst Perimeter
  • Worst Area & Mean Area
  • Worst Concave Points
  • Worst Texture

“Worst” values refer to the largest values across the tumor profile and tend to hold stronger predictive power.
